# Student-Result-Management-System
A Python-based Student Result Management System built using Jupyter Notebook to store, manage, and analyze student academic records.

## Problem Statement
Develop a Student Result Management System using Python that allows users to store, update, 
retrieve, and manage student academic records. The system should maintain details such as 
student name, roll number, subject-wise marks, total marks, percentage, and grade. This project 
aims to simulate a basic academic record system while reinforcing data handling and logical 
structuring skills. 

## Key Features
- Add student details such as name and roll number
- Enter and store subject-wise marks
- Calculate total marks and percentage
- Assign grades based on percentage
- View and update student result records
- Menu-driven and user-friendly interface

## Technologies Used
- Python
- Jupyter Notebook

## Steps to Run the Project
1. Open Jupyter Notebook
2. Open `student_result.ipynb`
3. Run all cells
4. Follow the on-screen menu options to manage student results
